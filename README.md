# Designer Portfolio

Веб-сервис для дизайнера-фрилансера: портфолио + система управления заказами.

## Описание

Сайт-платформа, совмещающая публичное портфолио и систему управления заказами с личными кабинетами клиентов.

## Технологии

- PHP 7.4+
- MySQL 5.7+
- HTML5, CSS3, JavaScript
- Bootstrap 5
